## Description

DNB Hub is a community-driven web application focused on the Portuguese Drum & Bass scene. The platform allows users to discover upcoming events and contribute information about artists and venues. This repository contains the **React frontend** of the application.

The app is built as a **Single Page Application (SPA)** and communicates with a REST API built with Express and MongoDB.

---

## Features

- Browse upcoming Drum & Bass events in Portugal
- View event details (date, city, venue, lineup)
- User authentication (signup / login / logout)
- Create, edit and delete events (authenticated users only)
- Add artists and venues (authenticated users only)
- Protected routes based on authentication status
- Responsive, mobile-first UI using pure CSS
- Loading states and error handling
- 404 Not Found page
